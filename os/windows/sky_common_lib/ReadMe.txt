=========================================================================
    확장 저장 프로시저 : sky_common_lib 프로젝트 개요
========================================================================


응용 프로그램 마법사에서 sky_common_lib.dll을 만들었습니다.  

이 파일에는 sky_common_lib 응용 프로그램을 구성하는 각각의 파일에
들어 있는 요약 설명이 포함되어 있습니다.

sky_common_lib.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    해당 파일을 생성한 Visual C++의 버전 정보를 비롯하여 
    응용 프로그램 마법사에서 선택한 플랫폼, 구성 및 
    프로젝트 기능에 대한 정보가 들어 있습니다.

sky_common_lib.cpp
    기본 DLL 소스 파일입니다.

proc.cpp
    이 파일에는 저장 프로시저 xp_proc이(가) 들어 있습니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h 및 StdAfx.cpp는
    sky_common_lib.pch라는 이름의 PCH(미리 컴파일된 헤더) 파일과
    StdAfx.obj라는 이름의 미리 컴파일된 형식 파일을 빌드하는 데 사용됩니다.


/////////////////////////////////////////////////////////////////////////////
기타 참고:

이 마법사를 마친 후 sky_common_lib.dll을 SQL Server의 
\Binn 디렉터리로 복사하십시오.

새 확장 저장 프로시저를 Visual Studio 데이터 프로젝트에서 추가하십시오. 
SQL Server 엔터프라이즈 관리자를 사용하거나 
다음 SQL 명령을 실행해도 추가할 수 있습니다.
  sp_addextendedproc 'xp_proc', 'sky_common_lib.DLL'

다음 SQL 명령을 사용하면 확장 저장 프로시저를 중지할 수 있습니다.
  sp_dropextendedproc 'xp_proc'

다음 SQL 명령을 사용하면 
서버에서 해당 DLL을 해제할 수 있습니다.
  DBCC xp_proc(FREE)
